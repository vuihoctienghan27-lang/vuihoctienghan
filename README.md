# vuihoctienghan

Nền tảng luyện thi TOPIK thông minh — Vui Học Tiếng Hàn.

## Công nghệ

- **Hosting**: Firebase Hosting
- **Auth / DB**: Firebase Auth (Google sign-in) + Firestore
- **CDN deps**: SweetAlert2, canvas-confetti, Pretendard/Inter fonts
- **Thuần**: Static HTML/CSS/JS, không framework, không build step, không package.json

## Chạy thử

```bash
firebase serve    # local dev
firebase deploy   # production
```
